# 雪国 – 二つの朝

> **歌詞登録ステータス：プレーン歌詞のみ**

```
君はさ無口で変わらない気がして
一つの夜を思い出してた
緩めた右足国道は長くて
あの日は雨が降っていたっけ
どこに、向かっているとかさわかんないけど
夜が明ける頃僕達は一つになる
それはさ私の未来の話
二つの朝を迎えるんだね
車はやさしく国道を走って
交わした意味のない会話を
どこで、変わってしまったとかもういいから
夜が明ける頃私達は過去になる
一粒、涙がこぼれた
見ていた朝日は何色に見えた?
最後、手を繋いでみる
僕らの想いは変わらずに届いた?届いた?
二粒、涙がこぼれた
見ていた朝日は2人のものじゃない
最後、手を繋ごうとか
変わらないあなたの横顔を照らした照らした
```
